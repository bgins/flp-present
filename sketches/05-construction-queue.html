Let C be a bivalent configuration of P, and let e = (p, m) be an event that is applicable to C. … Then, 𝒟 contains a bivalent configuration.

[ paper §3 ]
§ 3::the construction

Each stage consists of one or more process steps. The stage ends with the first process in the process queue taking a step in which, if its message queue was not empty at the start of the stage, its earliest message is received. This process is then moved to the back of the process queue.

[ paper §3 ]
▸ the queue + earliest-message rule is what makes the infinite run admissible — every nonfaulty process takes infinitely many steps, every message eventually gets delivered. The adversary's freedom is in WHICH stage each delivery lands.
§ 3::forever bivalent

Since each stage ends in a bivalent configuration, every stage in the construction of the infinite schedule succeeds.
