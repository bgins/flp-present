# flp-present

A visual walkthrough of the FLP impossibility result — Fischer, Lynch, and
Paterson's 1985 proof that no asynchronous consensus protocol can tolerate even
a single faulty process. Built with Svelte 5, Vite, and TypeScript.

Live (desktop): <https://bgins.github.io/flp-present/>

## Setup and run

```sh
npm install
npm run dev      # http://localhost:5173
```

## Build

```sh
npm run build    # static bundle in dist/
npm run preview  # serve the production build
```
